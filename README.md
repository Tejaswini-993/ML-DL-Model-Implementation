# ML-DL-Model-Implementation

## Overview

This repository contains implementations of core Machine Learning and Deep Learning models using Python. It covers fundamental concepts such as dimensionality reduction, neural networks, computer vision, and natural language processing using real-world datasets.

---

## Key Highlights

* Implemented PCA for dimensionality reduction
* Built CNN models for image classification (MNIST, CIFAR-10)
* Developed RNN (LSTM) for sentiment analysis (IMDB dataset)
* Applied Word2Vec and One-Hot Encoding for NLP tasks
* Performed data preprocessing and feature scaling using Scikit-learn

---

## Tech Stack

* Python
* NumPy, Pandas
* Matplotlib
* Scikit-learn
* TensorFlow / Keras
* Gensim

---

## Datasets Used

* MNIST (Digit Recognition)
* CIFAR-10 (Image Classification)
* IMDB (Sentiment Analysis)
* Wine Dataset (PCA)

---

## Results

* Achieved strong accuracy on classification tasks
* Visualized training vs validation performance
* Generated meaningful word embeddings using Word2Vec

---

## How to Run

```bash
pip install numpy pandas matplotlib scikit-learn tensorflow gensim
jupyter notebook
```
